Cuidados com a saúde

Dicas essenciais para manter uma vida saúdavel

Manter a saúde em dia é fundamental para uma vida equilibrada. Seguir algumas dicas simples pode ajudar muito no seu bem-estar diário.

Passos importantes

  1. Tenha uma alimentação balanceada.
  2. Pratique exercícios regularmente
  3. Beba bastante água

Alimentos saúdaveis

  • Frutas
  • Vegetais
  • Grãos Integrais
A saúde é o estado mais próximo da felicidade que conheço-Ditado popular

A hidratação é muito importante (e nunca deve ser negligenciada). Beba pelo menos 2 litros de água por dia para manter o corpo funcionando corretamente.

Exercícios como caminhada e corrida leve são ótimos para o coração. Evite alimentos processados que, muitas vezes, contêm conservantes. (Uma palavra como "natural" é muitas vezes usada para enganar consumidores desatentos.)

  • Comida muito gordurosa pode ser prejudicial.
  • Corrija hábitos antigos (como sedentarismo) e insira novos hábitos, como uma rotina de caminhada diária.
